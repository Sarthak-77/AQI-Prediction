# AQI Predictor — Frontend

React 19 + Vite single-page app for the AQI Predictor. See the
[root README](../README.md) for the full project overview, architecture, and
backend setup.

## Develop

```bash
npm install
npm run dev      # http://localhost:5173
npm run build    # production build
npm run lint     # eslint
```

The API base URL is set via `API_URL` in `src/App.jsx`.
